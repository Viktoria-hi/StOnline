# StOnline 🎓

**StOnline** — это современная веб-платформа для автоматизации учебного процесса. Электронный журнал, который объединяет студентов, преподавателей и администрацию в едином информационном пространстве.
## 🚀 О проекте
Проект разработан для упрощения учета успеваемости и предоставления быстрого доступа к учебной информации. StOnline позволяет забыть о бумажных носителях и обеспечивает прозрачность оценочной системы в реальном времени.

## ✨ Ключевые возможности

- **Личный кабинет студента:** просмотр оценок, посещаемости и расписания.
- **Рабочее место преподавателя:** быстрая выставление оценок, ведение журналов.
- **Панель администратора:** управление пользователями, группами и учебными планами.
- **Уведомления:** оповещения о новых оценках и изменениях в расписании.
- **Адаптивный дизайн:** удобная работа с ПК и мобильных устройств.

## 🛠 Технологический стек

- **Frontend:** React.js
- **Backend:** Python
- **База данных:** PostgreSQL
- **Другое:** Docker, Git
- **Аутентификация:** JWT (JSON Web Tokens)
- **Верстка/UI:** Tailwind CSS + shadcn/ui
- **Хостинг:** GitHub
